# ☕ Cafe Billing System (Python)

A simple command-line based cafe ordering and billing system built using Python.
This project allows users to select items from a menu, confirm their order, and generates a final bill with item-wise details.

---

## 🚀 Features

* 📋 Display cafe menu with prices
* 🛒 Add multiple items to order
* 🔁 Order confirmation (yes/no)
* 🔢 Quantity tracking using dictionary
* 💰 Automatic total bill calculation
* 🧾 Order summary (item + quantity + price)

---

## 🧠 Concepts Used

* Python Dictionaries
* Loops (while loop)
* Conditional Statements (if-else)
* User Input Handling
* String methods (`strip()`, `lower()`)
